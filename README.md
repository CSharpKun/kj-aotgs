In the process of creating the project, the following FOSS libraries were used:
- [EIRTeam/EIRTeam.FFmpeg](https://github.com/EIRTeam/EIRTeam.FFmpeg) GDExtension that allows to use any ffmpeg-supported video formats in Godot's video player, and has MIT license.
- Part of [FFmpeg](https://github.com/FFmpeg/FFmpeg) is used in EIRTeam's EIRTeam.FFmpeg GDExtension (such as `libavcodec`, `libavformat`, etc.), that allows to encode, decode and format video/audio/subtitles, and is licensed under GNU GPL 2.0, 3.0, LGPL 2.1 and 3.0 licenses.
